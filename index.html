Informasi Aplikasi

SI ONTEL adalah aplikasi kalkulator medis offline berbasis PWA (Progressive Web App) yang dirancang khusus untuk mempermudah tenaga kesehatan dalam menghitung pengenceran dosis obat untuk Syringe Pump dan Infus Pump secara presisi, cepat, dan aman.

Privacy Policy (Keamanan Data)

Aplikasi ini berjalan 100% OFFLINE pada browser Anda. Tidak ada data pasien, riwayat kalkulasi, atau pengaturan yang dikirim ke server internet. Semua data disimpan secara lokal menggunakan teknologi IndexedDB di perangkat Anda (Client-Side Storage).
