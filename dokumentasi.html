Bab 1: Pengenalan Aplikasi

SI ONTEL PRO adalah aplikasi web progresif (PWA) yang dirancang khusus untuk mempermudah perawat dan dokter dalam menghitung pengenceran serta titrasi dosis obat secara presisi, cepat, dan aman.

Aplikasi ini sangat cocok digunakan di ruang kritis seperti ICU (Intensive Care Unit) dan Kamar Operasi (OK) di mana dosis obat (seperti Norepinephrine, Dopamin, Dobutamin, dll) sangat dinamis dan membutuhkan penyesuaian instan terhadap mesin Syringe Pump maupun Infus Pump.

Keunggulan Utama: Aplikasi ini berjalan 100% Offline di perangkat Anda. Tidak memerlukan kuota internet setelah dimuat pertama kali, sehingga tidak akan terganggu oleh sinyal rumah sakit yang buruk.

Bab 2: Membaca Dashboard

Dashboard adalah layar pertama yang Anda lihat saat membuka aplikasi. Halaman ini berfungsi sebagai pusat kendali dan ringkasan aktivitas Anda.

  • Ringkasan Statistik: Menampilkan total kalkulasi yang pernah Anda buat, pembagian berdasarkan penggunaan alat (Syringe vs Infus), dan jumlah Master Obat yang tersimpan.
  • Grafik Top Obat: Grafik batang (Bar Chart) yang menunjukkan 10 obat apa saja yang paling sering Anda hitung. Sangat berguna untuk evaluasi tren pasien di ruangan.
  • Riwayat Terakhir: Menampilkan 3 perhitungan terakhir yang Anda simpan agar mudah diakses kembali jika pasien yang sama membutuhkan evaluasi.

Bab 3: Kalkulator Dosis (Fitur Utama)

Fitur ini digunakan untuk menghitung kecepatan alat secara instan berdasarkan target dosis tunggal yang diminta dokter.

Langkah-langkah Penggunaan:

  1. Pilih Metode Pemberian: Syringe Pump (suntikan otomatis) atau Infus Pump (tetesan makro).
  2. Pilih nama sediaan obat dari kotak pencarian. Data sediaan (mg) dan pelarut (cc) akan terisi otomatis.
  3. Masukkan Berat Badan Pasien (kg).
  4. Masukkan Dosis Target yang diinstruksikan oleh Dokter beserta satuannya (misal: mcg/kgBB/menit, mcg/menit, atau mg/jam).
Contoh Kasus Nyata

Seorang pasien di ICU (Berat Badan: 60 kg) diresepkan Norepinephrine dengan target dosis 0.05 mcg/kg/menit.

Input di Aplikasi:

  • Alat: Syringe Pump
  • Obat: Norepinephrine (Sediaan: 4 mg, Pelarut: 50 cc)
  • BB Pasien: 60 kg
  • Target: 0.05 mcg/kg/min

Hasil Otomatis Aplikasi:

  • Kecepatan yang di-set di alat: 2.25 cc/jam
  • Konsentrasi Obat: 80.0 mcg/cc

Di bagian bawah, aplikasi juga akan memunculkan Panduan Langkah Klinis berurutan (dari cara melarutkan hingga mengatur mesin) untuk mencegah malpraktik (medication error).

Bab 4: Generator Tabel Titrasi (ICU/OK)

Fitur ini adalah "senjata rahasia" bagi perawat di ruang kritis. Ketimbang menghitung kalkulator berulang-ulang setiap dokter mengubah dosis, Anda bisa mencetak tabel contekan (cheat sheet) khusus untuk satu pasien.

Cara Menggunakan:

  1. Masuk ke menu Tabel Titrasi.
  2. Pilih Obat dan masukkan Berat Badan Pasien.
  3. Aplikasi akan memunculkan deretan Range Dosis standar (misal: 0.01, 0.02, 0.03, dst). Anda bebas menambahkan atau menghapus angka di kotak tersebut sesuai kebutuhan rumah sakit.
  4. Klik tombol Buat Tabel.

Fitur Cetak & Bagikan:

Setelah tabel tercipta, Anda akan melihat dua tombol canggih:

  • Cetak Kertas: Mengubah layar HP menjadi mode cetak hitam-putih bersih. Menyembunyikan semua menu aplikasi sehingga Anda bisa mencetaknya via printer atau menyimpannya sebagai file PDF yang rapi untuk ditempel di tiang infus.
  • Simpan Gambar: Aplikasi akan otomatis mengubah tabel HTML tersebut menjadi file gambar (PNG) berkualitas tinggi dan membuka opsi Share. Anda bisa langsung mengirimkan gambar tabel tersebut ke Grup WhatsApp perawat atau menyimpannya di Galeri HP.

Bab 5: Manajemen Riwayat

Setiap kali Anda menekan tombol "Simpan Riwayat" atau "Ambil Gambar Label" di kalkulator, data tersebut masuk ke menu ini.

  • Pencarian Cepat: Anda bisa mengetik nama obat untuk mencari riwayat spesifik.
  • Tombol Reload (): Mengklik tombol ini akan mengembalikan seluruh angka (BB, dosis, target) ke formulir Kalkulator Dosis. Sangat berguna saat melakukan operan jaga (handover) ke shift berikutnya.

Bab 6: Master Obat (Database Lokal)

Aplikasi dilengkapi dengan 16 daftar obat darurat bawaan (seperti Epinephrine, Fentanyl, Propofol/Tiopol, dll). Namun, Anda memegang kendali penuh atas database ini.

Catatan Penting: Kebijakan pengenceran (Volume Pelarut) bisa berbeda di tiap Rumah Sakit. Contoh: Amiodarone mungkin dilarutkan dalam 20cc di RS A, tetapi dalam 50cc di RS B.

Di menu ini, Anda bisa menekan tombol Tambah Obat, atau menekan ikon edit pada obat yang sudah ada untuk menyesuaikan angka sediaan standar (mg) dan volume pelarut standar (cc) sesuai Standar Operasional Prosedur (SOP) tempat Anda bekerja.

Bab 7: Pengaturan & Backup Data

Personalisasi aplikasi sesuai keinginan Anda melalui menu Setelan.

Tampilan & Branding

  • Nama Aplikasi: Anda bisa mengubah "SI ONTEL" menjadi nama ruangan Anda (Misal: "Kalkulator ICU Mawar"). Akan langsung mengubah teks di seluruh aplikasi dan struk cetak.
  • Warna Tema: Ganti warna biru standar dengan warna favorit Anda.
  • Logo: Unggah gambar logo Rumah Sakit Anda agar tercetak di label dan tabel.

Backup dan Restore (Microsoft Excel)

Karena aplikasi ini berjalan 100% offline, jika Anda mengganti HP atau membersihkan cache browser, data Anda akan hilang. Untuk mengatasinya:

  • Klik Backup Data (Excel) untuk mengunduh file `.xlsx` yang berisi seluruh riwayat dan Master Obat Anda.
  • Di HP yang baru, cukup klik Restore Data (Excel) dan masukkan file tersebut. Semua data Anda akan kembali seketika!

Bab 8: Keamanan Data & Privasi

Berbeda dengan aplikasi kalkulator web pada umumnya, SI ONTEL dibangun menggunakan teknologi PWA dan penyimpanan IndexedDB (Client-Side Storage). Artinya:

  1. Data Anda tersimpan secara fisik di memori perangkat/HP Anda sendiri.
  2. Tidak ada satu byte pun data (baik itu pengaturan, riwayat, atau obat) yang dikirim ke server pusat, pihak ketiga, maupun cloud.
  3. Aman digunakan di lingkungan rumah sakit karena mematuhi privasi data klinis (meskipun aplikasi ini memang tidak meminta input nama pasien demi keamanan ekstra).
